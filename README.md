# YUKI YOGA LP

YUKI YOGA公式LINE登録を最優先CTAにした、クライアント確認用LPです。

## 公開前メモ

- `index.html`、`robots.txt`、`vercel.json` で noindex を設定しています。
- 画像はすべて `assets/` に配置しています。
- 料金表や詳細スケジュールは掲載せず、最新情報は公式LINEへ誘導しています。
- リンクや画像を差し替える場合は、HTML内のコメントと `assets/` のファイル名を合わせて更新してください。

## Vercel

GitHubにこのフォルダをアップロードし、Vercelでリポジトリを選択してデプロイします。フレームワーク設定は不要です。
